### Streszczenie
Projekt wyszukiwarki filmów został stworzony na potrzeby zaliczenia przedmiotu "Zaawansowane technologie internetowe". Jest to prosta wyszukiwarka filmów i informacji o nich na podstawie API udostępnionego przez [The Movie Database](https://developers.themoviedb.org/3/search/search-movies)

### Technologie
W projekcie zostały wykorzystane technologie takie jak:
- js-token: 4.0.0
- React: 16.13.1
- React-dom: 16.13.1
- jquery: 3.5.1
- React-scripts: 3.4.1

### Uruchomienie
Do uruchomienia tego programu najlepiej użyć Visual Studio Code. 
Aby uruchomić program należy otworzyć folder z aplikacją przy użyciu Visual Studio Code, następnie przy pomocy skrótu klawiszowego 
`Ctrl+~`
otworzyć konsolę.
W konsoli należy wpisać następujące polecenia:

```
npm install
npm start
```
Strona powinna otworzyć się automatycznie, jednak gdyby tak się nie stało, przejdź pod adres
```
localhost:3000
```
